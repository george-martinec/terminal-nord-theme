# Terminal Nord Theme

### How to install
1. Download file **terminal-nord-theme.txt**
2. Run command `dconf load /org/gnome/terminal/ < terminal-nord-theme.txt`

<br>

Command:
```
dconf load /org/gnome/terminal/ < terminal-nord-theme.txt
```
